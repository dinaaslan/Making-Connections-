# Making-Connections
For this assignment, I used an existing code I wrote for the “Profile” assignment and added a JavaScript file. For removing connection requests when clicking (✓/✖) buttons I used a function called removeRequests and added an onClick event to each button to remove the whole div that contains each connection request. 

For the second part of JavaScript, I added an onClick event on the (edit profile) element that calls a function called ChangeName. This function changes the element with the class of user-name and its innertext to another name.
